# what is browser?

A **browser** (web browser) is an application/software that helps you **access and view websites/pages** on the internet.

# What a browser does?
- Takes the **URL** you enter (example: https://example.com)
- Connects to the **web server**
- Downloads the web content (HTML, CSS, images, JavaScript)
- **Renders** the page and shows it on your screen

## Why it is used?
- To browse and read information online
- To watch videos, use web apps, and interact with websites
- To download and display web content safely and efficiently

## Types of browser (based on usage)
### 1) General-purpose browsers
Used for everyday web browsing.
- Google Chrome
- Mozilla Firefox
- Microsoft Edge
- Safari
- Opera

### 2) Search engine browsers
Designed around search results and web discovery.
- Example: Google Chrome (commonly used as a search-centric browser)

### 3) Mobile browsers
Made for smartphones/tablets.
- Chrome for Android
- Safari on iPhone/iPad
- Samsung Internet

### 4) Text-only browsers
Show pages without images/graphics (useful for low bandwidth).
- Lynx

### 5) Privacy-focused browsers
Focus on blocking tracking and improving privacy.
- Brave
- Tor Browser

### 6) Lightweight browsers
Smaller and faster for basic tasks (often for older devices).
- UC Browser (in some regions)

### 7) Specialized/enterprise browsers
Used for organizations with special security needs.
- Microsoft Edge for enterprise environments
