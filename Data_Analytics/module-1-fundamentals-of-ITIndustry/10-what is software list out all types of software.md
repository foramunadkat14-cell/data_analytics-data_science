# what is software ?

**Software** is a set of programs and instructions that tell a computer or device what to do. Without software, hardware cannot perform tasks.

---

## Types of software

### 1) System Software
System software manages and controls the computer hardware and provides a platform for other programs.
- **Operating System (OS)** (e.g., Windows, Linux, macOS)
- **Device drivers** (software that helps hardware like printer/keyboard work)
- **Utilities** (tools like antivirus, disk cleanup, backup tools)

### 2) Application Software
Application software helps users perform specific tasks.
- **Productivity tools** (MS Word, Excel, PowerPoint)
- **Browser & communication apps** (Chrome, Firefox, Zoom)
- **Media & design apps** (Photoshop, VLC, music players)
- **Database software** (MySQL, Oracle, MongoDB)
- **Educational software** and many more

### 3) Programming / Development Software
Development software is used to create other software.
- **Programming languages** (Python, Java, C++)
- **IDE (Integrated Development Environment)** (VS Code, Eclipse)
- **Compilers / Interpreters** (convert code into machine-understandable form)
- **Frameworks** (Django, Spring)

---

## Other common classification (by user type)

### 1) Freeware
Software available for free (may have limited features).

### 2) Shareware
Software that can be tested for a time, then requires payment.

### 3) Open-source software
Source code is available to everyone; users can modify and improve it.

### 4) Proprietary (Closed-source) software
Source code is not shared; it is owned by a company.

---

## Summary
Software is the set of programs/instructions that control and run tasks on computers. Common types are:
- **System software** (OS, drivers, utilities)
- **Application software** (Word, browsers, media apps, databases)
- **Programming software** (languages, IDEs, compilers, frameworks)

You can also classify software as **freeware, shareware, open-source, and proprietary**.
