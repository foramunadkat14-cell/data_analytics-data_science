# what is excel ?

Excel is a **spreadsheet application** by **Microsoft** used to **store, organize, calculate, and visualize data**.

## Why Excel is used
- Maintain data in **rows and columns**
- Do **calculations** using formulas
- Make **charts/graphs** for easy understanding
- Help in **data analysis** (sorting, filtering, pivot tables)

## Basic parts of an Excel sheet
- **Workbook**: The full Excel file (can contain multiple sheets)
- **Worksheet**: A single page inside a workbook
- **Row**: Horizontal line of cells
- **Column**: Vertical line of cells
- **Cell**: The intersection of a row and a column (example: A1, B2)

## Example structure (table in Excel)
Excel is commonly used like a table:
- columns = fields (Name, Salary, Date, etc.)
- rows = records (each employee/customer/data item)

## Common Excel features
- **Formulas**: +, -, SUM(), AVERAGE() etc.
- **Functions**: built-in operations like statistical and financial functions
- **Formatting**: better readability (colors, borders, number formats)
- **Sorting & Filtering**: find required data quickly
- **Charts**: bar chart, line chart, pie chart, etc.

## Excel in Data Analytics
Excel helps analysts by:
- cleaning and organizing data
- performing calculations
- summarizing data
- visualizing results to make decisions
