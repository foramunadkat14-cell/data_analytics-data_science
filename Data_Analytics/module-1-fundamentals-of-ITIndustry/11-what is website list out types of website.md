# what is website ?

A **website** is a collection of web pages (and related files like images, videos, and scripts) that are accessed through the internet using a **URL**.

Websites are hosted on servers (web servers) and viewed using web browsers.

---

## Types of websites (common categories)

### 1) Personal / Portfolio Website
- Used by individuals to show skills, work, and personal info
- Example: portfolio site of a designer/developer

### 2) Business / Corporate Website
- Used by companies to describe products/services and contact details
- Example: company official website

### 3) E-commerce Website (Online Shopping)
- Used to buy and sell products/services online
- Includes cart, payments, order tracking
- Example: Amazon, Flipkart

### 4) Educational Website
- Used for learning and training
- Example: course sites, tutorials, online classes

### 5) Blog / News Website
- Used to publish articles and updates
- Often includes categories, comments, and RSS feeds

### 6) Social Media Website
- Platforms for user interaction and sharing content
- Example: Facebook, Instagram, LinkedIn

### 7) Government Website
- Official websites for government departments and services
- Provide information and online forms/alerts

### 8) Entertainment Website
- Provides videos, games, streaming, and fun content
- Example: YouTube (video), gaming portals

### 9) Non-profit / NGO Website
- Used by organizations to raise awareness and collect donations

### 10) Forum / Community Website
- Users discuss topics, ask questions, and share knowledge
- Example: Reddit-like communities

---

## Summary
A **website** is a set of web pages accessible via a URL. Common types include:
- personal/portfolio
- business/corporate
- e-commerce
- educational
- blog/news
- social media
- government
- entertainment
- non-profit
- forum/community
